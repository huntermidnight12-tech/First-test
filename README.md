# تطبيق مقارن الأسعار بالباركود

## الفكرة
تطبيق ويب مخصص للآيفون يسمح لك بـ:
- إدخال اسم المنتج
- إدخال مكان البيع
- إدخال السعر
- ربط كل ذلك بالباركود
- مسح الباركود بالكاميرا الخلفية
- عرض كل الأسعار لنفس المنتج وترتيبها من الأرخص إلى الأغلى

## ملاحظات مهمة
- الكاميرا تعمل فقط عبر HTTPS أو localhost.
- البيانات محفوظة محلياً داخل المتصفح على نفس الجهاز.
- يمكن تصدير واستيراد البيانات بصيغة JSON.
- يستخدم مكتبة html5-qrcode.

## طريقة التشغيل
1. ارفع الملفات على استضافة HTTPS أو GitHub Pages.
2. افتح الصفحة من Safari على iPhone.
3. عند أول ضغط على زر المسح اضغط "سماح" للوصول إلى الكاميرا.
4. يمكن بعد ذلك إضافة الصفحة إلى الشاشة الرئيسية.
